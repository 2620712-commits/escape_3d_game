# 3D 방탈출 게임 🚪

집 안에 갇혀있고, 방 문을 열고 탈출하는 게임입니다!

## 🎮 게임 설명

### 목표
- 집 안의 **3가지 퍼즐**을 풀어 숫자를 찾기
- 찾은 숫자 3개를 조합해 **비밀번호** 풀기
- 방 문을 열고 **탈출하기**

### 게임 구조

**방의 배치:**
- 중앙 거실 (플레이어 시작점)
- 퍼즐 1: 좌측 뒷벽 (수학 퍼즐 - 숫자 획득)
- 퍼즐 2: 우측 뒷벽 (그림 맞추기 - 숫자 획득)
- 퍼즐 3: 왼쪽 앞 (숨겨진 상자 - 숫자 획득)
- 탈출문: 앞쪽 중앙 (자물쇠 - 3자리 비밀번호 필요)

## 🕹️ 조작 방법

| 키 | 기능 |
|---|---|
| **W** | 앞으로 이동 |
| **A** | 왼쪽 이동 |
| **S** | 뒤로 이동 |
| **D** | 오른쪽 이동 |
| **마우스** | 카메라 회전 (시점 조작) |
| **E** | 퍼즐 풀기 / 문 열기 |
| **ESC** | 게임 종료 |

## 🧩 퍼즐 시스템

### 퍼즐 1: 수학 퍼즐
- 위치: 방의 왼쪽 벽
- 문제: 간단한 수학 계산
- 정답: 0~9 사이의 숫자

### 퍼즐 2: 그림 맞추기
- 위치: 방의 오른쪽 벽
- 문제: 모양 패턴 맞추기
- 정답: 0~9 사이의 숫자

### 퍼즐 3: 숨겨진 물건
- 위치: 방의 왼쪽 앞 (주황색 상자)
- 문제: 숨겨진 상자 찾기
- 정답: 0~9 사이의 숫자

## 🔐 비밀번호 입력

1. 세 퍼즐을 모두 풀면 각 퍼즐에서 얻은 숫자가 기억됩니다
2. 방의 앞쪽 중앙에 있는 문으로 이동 (황금색 자물쇠)
3. **E키**를 눌러 비밀번호 입력 모드 시작
4. **숫자 키**로 3자리 비밀번호 입력 (0~9)
5. **Enter**를 눌러 확인
6. 정답이면 문이 열리고 탈출!

## 📋 설치 및 실행

### 필수 요구사항
- Python 3.7 이상
- pygame
- PyOpenGL
- numpy

### 설치 방법

```bash
cd escape_3d_game

# 2. 라이브러리 설치
pip install -r requirements.txt
```

### 게임 실행

```bash
python escape_room_3d.py
```
